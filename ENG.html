Projects

Cats vs Dogs

Another project in data analysis and visualisation. This time I did the analysis in Excel, but the data was visualised in PowerBI. The dataset came from Kaggle. From this dataset I wanted to extract the following information:
1. which states prefer cats and which states prefer dogs?
2. which states have the most dogs and cats per household?
3. which states have almost the same number of cats and dogs?
All calculations were done on separate Excel sheets so that the progress of the work could be tracked and, if there were any errors, they could be quickly found and corrected. The project file is posted on GitHub.
Potential use of the analysis: when deciding to build veterinary clinics in the USA or pet shops, running special events for dog and cat owners.

Cars

My first project in data analysis. Data cleaning and analysis was done in Excel, as well as basic visualisation. The dataset for visualisation was downloaded from Kaggle. When analysing the data, I wanted to answer the following questions:
1. which companies produce the fastest cars?
2. What are the 3 fastest cars?
3. What is the average acceleration of the cars of one company?
4. which cars have the highest number in the set?
All calculations were done on separate Excel sheets, so that the progress of the work can be tracked and if there are any errors, they can be found and corrected quickly. The project file is posted on GitHub.
